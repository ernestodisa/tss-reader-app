## Gotchas duros (aprendidos a golpes)

- **CSS legacy**: no re-agregar clases `.reader-*`/`.library`/`.dropzone` a `global.css` — duplicadas pisaban los media queries (ya purgado; ver banner en global.css).
- **Zustand + React 19**: selectores que devuelven arreglos/objetos nuevos por llamada → loop #185 y pantalla en blanco (fix pattern en `AnnotationsPanel.tsx`: suscribirse a arreglos crudos + `useMemo`).
- **TOML de wrangler**: claves sueltas después de una tabla `[[...]]` se las traga la tabla (así se fugó `workers_dev=false` una vez).
- **wrangler multi-versión**: raíz usa wrangler 4 (exige `wrangler login` fresco si el refresh token muere), `worker/` usa wrangler 3. Si aparece "user auth missing api token non interactive", Ernesto debe correr `npx wrangler login` en SU terminal.
- **Verificación en el panel**: el service worker cachea agresivo — antes de verificar un deploy: unregister SW + purge caches + reload (snippet usado repetidamente en la sesión).
- El screenshot del panel a veces comprime el layout en una banda — confiar en mediciones DOM (`getBoundingClientRect`), no en el pixel del screenshot.
